# Running Away From The Rain

**Running Away From The Rain** is a simple arcade-style game developed with Python and Pygame. The player controls a character that must avoid falling raindrops for as long as possible. As time passes, the game becomes more challenging by increasing the frequency of falling obstacles. This project demonstrates core game development concepts such as event handling, collision detection, object movement, and score tracking using Pygame.
## Installing
Download the Python installer package from the official website and install it, if not installed previously.

Run the following in the terminal to install the Pygame library
```
pip install pygame
```

## Running the application
Download the source code from the repository and run the file.
```
python main.py
```
